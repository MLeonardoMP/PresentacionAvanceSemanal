# Presentación Avance Semanal — CAC

Slides semanales de avance por sub área de la **Coordinación de Tecnología (CAC)**.
Presentación estática en HTML/CSS puro, sin frameworks ni JavaScript.

## Versiones

| Ruta | Layout | Descripción |
|---|---|---|
| `/` | Grid 2×2 | 4 cards, una por sub área |
| `/v2` | 3 cards centradas | Layout alternativo |

## Sub áreas

| Color | Sub área |
|---|---|
| Azul | Pruebas, Documentación y Soporte |
| Magenta | Base de Datos |
| Verde | Infraestructura |
| Ámbar | Desarrollo |

## Stack técnico

- HTML5 semántico + CSS inline (un archivo por semana)
- Sin JavaScript, sin frameworks
- Font: [Inter](https://fonts.google.com/specimen/Inter) (Google Fonts)
- Dimensiones del slide: **1000px × 580px**
- Dark aesthetic — paleta de grises oscuros con acentos de color por área
